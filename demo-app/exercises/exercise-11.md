GitHub [https://github.com/t4d-classes/react-redux-dotnet-core_02032020](https://github.com/t4d-classes/react-redux-dotnet-core_02032020)

## Steps

1. Beneath the buttons in Calc Tool, display a history of the operations performed. For each operation record the type of operation and the value. Do not display results.

2. Next to each operation place a delete button. When the button is clicked remove the operation from the list.

3. Add a clear button, when the button is clicked, clear the history and reset the result to 0.

Bonus:

4. When clicking the delete button for a history item, update the result to be the value it would be if the history item had never existed.
